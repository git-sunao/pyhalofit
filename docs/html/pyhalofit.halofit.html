set_pklin(k, pklin, z, unit='h/Mpc')¶

Setting Fourier mode and linear matter power spectrum at desired redshift.

Parameters

  • k (array_like) – Array of Fourier modes

  • pklin (array_like) – Array of linear matter power spectrum at redshift z

  • z (float) – Cosmological redshift

  • unit (string) – Unit of Fourier mode. ‘/Mpc’ and ‘h/Mpc’ are available. Default is ‘h/Mpc’.
